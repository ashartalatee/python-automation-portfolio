 Deskripsi Proyek
Data Cleaner Otomatis adalah aplikasi Python yang bisa:

- Membersihkan data Excel/CSV secara otomatis
- Menghapus row kosong
- Menghapus whitespace di kolom dan sel string
- Mengisi nilai kosong di kolom numerik dengan 0

Tujuannya: **menghemat waktu admin** dan meminimalisir kesalahan manual.

---

 Fitur
| Fitur | Deskripsi |
|-------|-----------|
| Upload File | Excel (`.xlsx`) atau CSV (`.csv`) |
| Preview | Lihat data sebelum dan sesudah dibersihkan |
| Download | Hasil bersih bisa di-download langsung |
| Automation | Semua langkah otomatis tanpa coding tambahan |
